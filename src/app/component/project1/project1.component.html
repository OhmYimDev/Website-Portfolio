Project Overview

Attraction paper is an Android mobile application built from Ionic framework that works with Angula framework, the application is connected to the phpmyadmin database using sql and php to manage the database. The features include Subscriptions, Logins, Check Ins, Rating Reviews, Location Map, Writing New Attractions and ranking of popular tourist attractions.

I was inspired by similar platforms like Wongnai for example and thought it would be a fun project to make my own application, in the beginning I only had a little knowledge of html css php and sql.
