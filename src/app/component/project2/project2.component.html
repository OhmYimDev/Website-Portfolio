Project Overview

Barfery Days is a website I created for fun. and put it into practice in the Barfery Days Shop.

I work part-time at Barfery days shop. I am responsible for managing products in the shop such as packing parcels, Check the amount of inventory, Check the amount of raw materials remaining, etc.

Checking inventory every time an order arrives at the store is tedious. I can't figure out how many pieces are left. So I thought it was easy to create a website. it's fun where I can save the number of every item in the shop, can add, delete, edit at any time. And every store employee can log in to view this information from anywhere.

It became a Barfery Days website with simple functionality using PHP CRUD connected to phpmyadmin for data management. Website operations such as adding, removing, modifying the number of products. Show product management history By displaying the product management list, date, time, etc., making it convenient to check products every day.
