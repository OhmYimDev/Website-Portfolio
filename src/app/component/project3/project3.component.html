Project Overview

I think the best way to learn is to do front-end developer projects and learn along the way.

Starbucks landing page clone is my case study. build the homepage of Starbucks.com from scratch with HTML, CSS and a bit of JavaScript

Starbucks landing page clone is my case study. create home page Starbucks.com From scratch with a little HTML, CSS and JavaScript, it was exciting for me to build a world-class brand website. I've learned how to structure html, create css classes where classes are applied to multiple elements at the same time, use display grid or flexbox.

My next challenge was to create a responsive website. I learned how to optimize the layout and size for each screen size. I'm a bit confused as to what layout to use. or how much to resize in different screen sizes But now I've learned it.
